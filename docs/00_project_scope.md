# Project Scope

## Scientific Focus

This project focuses on target-site resistance to PPO-inhibiting herbicides in *Amaranthus* species, especially *Amaranthus tuberculatus* and *Amaranthus palmeri*.

The repository is currently limited to evidence curation, metadata schemas, and reproducible validation checks. Specific mutations, residue positions, accession numbers, and biological conclusions must only be added when supported by citation metadata.

## Eventual Dashboard Goals

The dashboard will eventually organize curated information about PPO gene and protein sequence metadata, verified resistance mutation evidence, herbicide metadata, 3D PPO structure models, docking results, and evidence levels.

Experimentally confirmed resistance evidence will be separated from computational predictions. Computational structural models and docking results will be labeled as prediction-based evidence and will not be treated as proof of field resistance.

## Currently Excluded

The current setup excludes dashboard application code, downloaded papers, stored PDFs, generated model files, docking result files, and uncited mutation records.

The repository should store curated links, citation metadata, accession metadata, mutation evidence tables, and reproducible scripts only.

## Evidence Hierarchy

1. Experimentally confirmed mutation and phenotype.
2. Peer-reviewed reported mutation.
3. Computational structural prediction.
4. Docking prediction.
5. Unverified claim.
