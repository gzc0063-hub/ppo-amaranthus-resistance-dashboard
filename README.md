# PPO Inhibitor Resistance Dashboard for Amaranthus spp.

This repository is the early project setup for a reproducible research dashboard focused on PPO target-site resistance in *Amaranthus tuberculatus* and *Amaranthus palmeri*.

The project is currently in an evidence curation stage. Its first priority is to define careful schemas, citation metadata, and reproducible checks before adding dashboard code or scientific interpretation.

The dashboard will eventually connect curated PPO gene and protein sequence metadata, verified resistance mutations, herbicide information, 3D PPO structure models, docking outputs, and evidence levels. Experimentally confirmed resistance evidence must remain clearly separated from computational predictions.

Paper PDFs are not stored in this repository. The repository should store only curated citation metadata, links, accession tables, mutation evidence tables, and reproducible scripts.

Docking predictions are hypothesis-generating only. Docking scores must never be treated as proof of field resistance.
